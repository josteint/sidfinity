# Lemon64: BZ Utils / Rob Hubbard Editor Thread

Source: https://www.lemon64.com/forum/viewtopic.php?t=8111
Date: June 2003 - January 2011

## BZ Utils Tape

Dan Gillgrass (June 17, 2003) mentioned having the editor on cassette from
the BZPD range. The utility was called "BZ Utils tape."

## Zoolook-Based Editor

TMR (June 17, 2003): The editor was "built around Hubbard's Zoolook tune by
Huddy and Greeny." He described it as "a little primitive" but noted that
"almost any current driver can handle the sounds."

## Hubbard's Workflow

TMR confirmed: Hubbard himself "edit[ed] the data directly into the source
code." He never used a graphical editor -- he composed by typing hex data into
the Mikro Assembler.

## Red/Judges Driver Conversion

TMR mentioned that Red/Judges (Jeroen Kimmel) "converted the driver back into
source and used that for some of his stuff."

groepaz (January 4, 2011) corrected: the converted driver was specifically
from **Crazy Comets**, not Commando as sometimes stated.

## Ace 2 Player Editor

groepaz referenced "another editor, based on the 'ace 2' player" and provided
a CSDB link: https://csdb.dk/release/?id=75124

logikstate (January 2011) created an editor/ripper tool demonstrating Ace 2
player compatibility with the ability to load multiple song formats.

## Technical Content

The thread itself contains minimal byte-level technical information. The
useful takeaways are:
1. Multiple people have built editors around Hubbard's driver format
2. The Crazy Comets driver was converted to editable source by Red/Judges
3. The Ace 2 player is a related/compatible format
4. The CSDB release at id=75124 may contain more technical details
