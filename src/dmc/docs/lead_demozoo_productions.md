## DMC V6.0 — Technical Details (most important for engine support)

**CSDb:** https://csdb.dk/release/?id=170999
**Released:** November 4, 2018
**Developer:** The Imperium Arts (Brian + The Syndrom)
**Documentation:** http://csdb.dk/getinternalfile.php/174455/dmc6-docs.txt (600 downloads)
**Disk image:** http://csdb.dk/getinternalfile.php/174454/dmc%20version6.D64 (453 downloads)
**Soundtrack SID:** `/MUSICIANS/T/The_Syndrom/DMC_V6_note.sid` (HVSC)

### Technical notes from documentation
- Editor has Track Edit Mode, Sector Edit Mode, Sound Editor
- **Wavetable fields:** AD (attack/decay), SR (sustain/release), WV (pointer to wave table), P1/P2 (pulse values), V1 (Vibrato), F1/F2/F3 (Filter settings)
- **Three dedicated tracks:** vibrato channel, filter channel, drum channel (fixed frequency notes)
- **Commands in sector edit:** SND, DUR, END, pause
- **DUR commands must always be followed by a note** (important player constraint)
- **Speed setting:** controlled via memory location (example given: `$10e9 lda#$01`)
- Speed settings require manual adjustment post-packing
- Keyboard commands: F1/F3/F5/F7 function keys, Shift-C (color), Shift-T (multispeed type), Shift-V (volume/speed/filter), Ctrl-C/L/S/D (disk ops)
- Track editing: buffer copy/paste, sector editing, "switch to tune2 (f.e. to work on individual sectors)"
- **Low raster time** — noted by multiple commenters as a key feature

### User comments (verbatim from CSDb)

**Key insight:** V6 was "talked about a lot back in the nineties" (Jazzcat) — it was a long-delayed release that finally shipped in 2018. Very low raster time is a design goal. Songs made with V6 will be in HVSC under `/MUSICIANS/T/The_Syndrom/` and potentially others.

## Rare/Unusual Variants Not in HVSC (as player tools)

1. **DMC V6.0** (2018) — Latest version, only 453 downloads, probably not in HVSC player identification. Player format differs from V5 (very low raster time per comments). Documentation available at CSDb.

2. **DMC 4 25Hz** (2020, MultiStyle Labs) — Patched to run at 25Hz (PAL half-speed). Entirely post-HVSC. Songs made with this tool will have different timing than standard DMC4.

3. **DMC V5.01B** (Chaos/Zeux) — Bug-fixed V5.0, only 78 downloads. May have subtle player differences from V5.0.

4. **DMC V4.0 Pro Tool Kit** (Onslaught, Morbid only) — Only 115 downloads. Not the same as "4.0 Professional" (Graffity+Onslaught). Rare standalone toolkit.

5. **DMC 4.G** (1997, Samar/Glover) — Samar-specific fork, only 497 downloads. Glover contributed extra code, so player may differ slightly from other DMC4 variants.

6. **The Superiors DMC V2.1+** — Only broken versions found as of 2024. Original only in Music Toolz #2. If intact songs exist, player identification would be difficult.

7. **DMC v4.0+ (Tengu/Dow Jones)** — Tengu-group fork from 1995, separate from all Onslaught variants. Only in Music Toolz compilation.

## Connections Relevant to SIDfinity DMC Parser

1. **DMC V4 vs V5 are the dominant HVSC variants.** V3 and earlier are rare. V6 is too new (2018) to have much HVSC presence but may have SIDs in HVSC under The Imperium Arts members.

2. **The "x", "y", "G" version letters** in Onslaught releases (4.X, 4.y, DMC 4.G) indicate iterative patches. The 4.G "G" is specifically Glover's contribution. These likely have minor player differences but the same fundamental song format.

3. **25Hz variant** changes playback rate from 50Hz to 25Hz — this affects tempo/frame timing. Any songs from this tool will play at half-speed if played with the standard 50Hz DMC player.

4. **Low raster time in V6** is a confirmed design goal (multiple independent CSDb comments). This suggests V6 uses a simplified/optimized player loop vs V5.

5. **DUR command must always be followed by a note** (from V6 docs) — this is a parser-relevant constraint for the sector/pattern data format.

6. **Wavetable fields in V6:** AD, SR, WV (pointer), P1/P2 (pulse), V1 (vibrato), F1/F2/F3 (filter) — these column names are from the editor UI, not necessarily identical to the binary layout but useful for cross-referencing.
