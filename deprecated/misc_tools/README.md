# Miscellaneous Tools

Standalone utilities from various development phases. Some are useful for debugging but aren't part of the active pipeline.

- `analyze_player.py` — analyze SID player behavior from siddump write logs
- `audio_compare.py` — compare audio waveforms (FFT-based, experimental)
- `gt2_batch_test.py` — ad-hoc batch testing (superseded by regression_test.py)
- `gt2_tempo.py` — DEFAULTTEMPO extraction (merged into gt2_decompile.py)
- `validate_data.py` — compare data sections of original vs rebuilt GT2 SIDs
