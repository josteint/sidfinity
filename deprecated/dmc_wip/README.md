# DMC Work-in-Progress

Early attempt to transpile DMC (Demo Music Creator) tunes to GoatTracker format. This approach was abandoned in favor of DMC → USF direct conversion.

- `dmc_to_gt2.py` — DMC sector → GT2 packed pattern transpiler

The active DMC files (`src/dmc_parser.py`, `src/dmc_to_usf.py`) are still in src/ for future DMC support work.
